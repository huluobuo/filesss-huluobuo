---------------------------------文件加解密工具---------------------------------

#### 1.0.1版本
- 生成密钥
- 加密文件
- 解密文件

## OTHER:
    实现了文件加解密工具，可以生成密钥，加密文件，解密文件。
    代码中使用了`pycryptodome`库来实现AES加密算法。

#### 1.0.2版本
- 生成密钥
- 加密文件
